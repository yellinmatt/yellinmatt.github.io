What this is: the most spatial move. The animated "Stand-up's greatest hits, on one page" word-swap line gets removed; the search input takes its place — a single horizontal yellow underline, Archivo uppercase as you type, magnify on the left, "/" key on the right. The input replaces the marketing line. Trade-off: you lose the animated swap tagline (which carries voice but is decorative) and gain a search input that lives in the most prominent slot the masthead has.
  • Pro: highest semantic priority — the search is the masthead's primary call to action, not just an element in it; least visual mass added (no box, no fill)
  • Con: kills the kinetic word-swap moment; the placeholder has to carry the marketing voice instead
Header consistency. Whichever B you pick, the inner-page header strip uses the same visual grammar miniaturized: same border, same magnify treatment, same color. The "/" key still focuses it from anywhere. The current "⌖ Search" pill goes away.

Position note. B1–B4 all sit between the word-swap line and the comic-of-the-day chips (the baseline B position). B5 replaces the word-swap line entirely — different spatial bet.
