What this is: the Index band, which currently shows just stats and a "Browse →" CTA, becomes a full search surface. Title + subtitle on top, full-width search input in the middle, stats inline as labels (no separate count column). The CTA goes away; the search IS the way you browse the Index.
  • Pro: consolidates two ideas (search + Index entry point) into one; the Index band stops being passive; the search inherits authority from the dark black band aesthetic
  • Con: kills the current count-up animation moment; loses the at-a-glance stats column
  • Vibe: the catalogue's command line

What this is: the home page top stays as it is (masthead untouched). As you scroll past the masthead, a tall yellow-bordered search bar slides down under the nav and stays pinned. Always one keystroke away once you start browsing. The "/" key focuses it from anywhere.
  • Pro: non-disruptive at first impression; becomes maximally available the moment a user is in browsing mode
  • Con: first-time visitors miss it until they scroll; adds a sticky element competing with the existing nav
  • Vibe: Google-style persistence; ambient utility rather than headline feature
