Notes on the inner-page header treatment.
Whichever C you pick, the inner-page header search strip uses the SAME visual language miniaturized: same border, same magnify treatment, same color. The current ⌖ Search pill goes away; an always-visible input takes its slot. The "/" key still focuses it from anywhere.

Want a hybrid? Tell me letters + adjustments — e.g., "C2 for the home card, C1 for the inner-page header" or "C3 but make the stamp red instead of yellow."
