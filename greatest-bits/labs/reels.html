Reels.

A working preview of the Instagram-reel embed pattern for Greatest Bits bit cards. Real cards, real embeds, real schema. Approve here, then it rolls out.

1 / Schema

One field added. Default stays YouTube.

The current bit-object gets one optional src key. When it's omitted or set to "youtube", every existing bit renders exactly as today. When it's "instagram", the bit-player swaps the YouTube iframe for an Instagram-reel iframe and reads the shortcode instead of a video id.

// existing YouTube bit (unchanged — no migration needed)
{ t, subj, sp, yr, id }
// extended — src defaults to "youtube" when omitted
{ t, subj, sp, yr, id, src: "youtube" }
// NEW — Instagram reel bit
{ t, subj, sp, yr, src: "instagram", shortcode: "DRLq4H_jPiW" }

4 / Tradeoffs

What you give up, what you gain.

Reels are how a growing share of contemporary stand-ups distribute their best short-form material. The catch is the verification gate that YouTube oembed gives us for free doesn't exist on Instagram, so each reel-bit needs a curator-level pick instead of a machine-verified one.

YouTubeInstagram reel
Verificationoembed API · author + title returnedno oembed · manual visit, curator confirms
Embed shape16:9 horizontal iframe9:16 vertical iframe (max 420px wide)
Stabilitygood · official channels rarely deletepoor · creator can delete a reel anytime
Title sourceYT title we match against `t`no title field · curator names the bit
Rights posturesanctioned by official channelsanctioned by account holder
Audit-game flagfull chip set (wrong clip / title / etc.)"reel removed" / "wrong attribution"
5 / Pilot

Five comics first. Then open the door.

  1. Land schema + bit-player branch on the three rendering surfaces: audit.html (already done), comic.html, special.html.
  2. Add one canonical reel each for five IG-first comics: Maddy Kelly (Quiet Piggy), Hannah Berner, Patti Harrison, Catherine Cohen, plus one Matthew picks.
  3. Verify each reel renders inline on the live comic page; confirm the audit-game IG flag chips behave.
  4. Open the door to bulk-adding reel-bits across the catalogue's modern comics — IG-first creators are now in-scope for top-N consideration.
  5. Add an "IG reel" filter facet on the Index so reel-bits surface visibly.
