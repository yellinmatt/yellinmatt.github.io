How it's built

  1. Discover broadly

    The roster fills out wide first, across all eras and media, living and dead, so judgment works on a full board instead of a blank page.

  2. Verify every link

    Each clip is checked for embeddability and an official source, and each website and channel link has to resolve before it ships. A dead or re-uploaded link gives way to an honest "coming soon."

  3. Flag the gaps

    Where a bit matters but no official clip or sourceable image exists, the gap is shown plainly. Nothing is ever fabricated.

Honest caveats

The list order is seeded and provisional: a starting point from primary sources, not the final ranking. Some specials still show a "coming soon" poster where none has been sourced, and a few comics carry an honest gap where no official, embeddable clip exists. The catalogue grows and corrects continuously. The full rulebook for sourcing and verification is in Method & Sources.
